how does javascript work

javascript is a client side scripting language.Client-side scripting language means that it runs at the client-side( or on the client machine) inside the web-browsers.JavaScript is considered high-level because it does not require direct interaction with the operating system and hardware.JS can be used for declarative (functional) or imperative (object-oriented) programming styles.JS is dynamic weakly typed language, meaning you do not annotate variables with types (string, int, etc) and the true types are not known until runtime.JS can only run one instruction at a time, even if your CPU has multiple cores and available threads.

what difference between local storage and session storage

localStorage and sessionStorage is similar ; the difference is that while data in localStorage doesn't expire but data in sessionStorage is cleared when the page session ends.That is, the data stored in localStorage persists until explicitly deleted. Changes made are saved and available for all current and future visits to the site. For sessionStorage, changes are only available per tab. Changes made are saved and available for the current page in that tab until it is closed. Once it is closed, the stored data is deleted.So we say that, localStorage - use for long term use. sessionStorage - use when you need to store somthing that changes or somthing temporary
